# Opoplatform

Material de preparacion para oposiciones TIC de la Administracion General del Estado.

Rutas principales:

- `funcionario.txt`: indice maestro y contexto general.
- `fuentes_actualizadas_2026.txt`: fuentes BOE/INAP y normativa base.
- `calendario_12_semanas_a1_a2.txt`: plan inicial de estudio.
- `banco_test_inicial_100.txt`: primer banco de preguntas.
- `banco_supuestos_a1_a2.txt`: supuestos practicos A1/A2.
- `plantillas_respuesta_examen_a1_a2.txt`: plantillas para responder con estructura de tribunal.
- `respuestas_modelo_desarrolladas.txt`: respuestas modelo desarrolladas.

Carpetas:

- `opo1/`: ruta C1 TAI.
- `opo2/`: ruta A2 GSI.
- `opo3/`: ruta A1 TIC.

Regla de trabajo:

Antes de cerrar cualquier contenido juridico o de convocatoria, contrastar con BOE/INAP y dejar la fuente en `fuentes_actualizadas_2026.txt`.
