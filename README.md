## Indice maestro

- `funcionario.txt`: indice maestro y contexto general.
- `fuentes_actualizadas_2026.txt`: fuentes BOE/INAP y normativa base.
- `seguimiento_convocatorias_2026.txt`: estado vivo de convocatorias.
- `mapa_produccion_material.txt`: hoja de ruta de produccion del material.
